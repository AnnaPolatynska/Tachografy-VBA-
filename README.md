# Tachografy-VBA-
Zlecenie z labolatorium tachografów Głównego Urzędu Miar.
Cel: naniesienie poprawek w działaniu istniejącej bazy danych MS ACCESS tachografów analogowych i cyfrowych (VBA).

Zapisana wersja bazy ma zatarte dane osobowe, ma pokazywać jedynie sposób działania poszczególnych jej kompotnentów.

Opis: 
W istniejącej bazie danej MS Accesss należy dodać przyciski i pola umożliwiające jej dostosowanie do nowych przepisów.
Należy usunąć nadmiarowe kwerendy zachowując prawidłowe działanie bazy, ułatwić poruszanie się po formularzach i wydrukach 
i zaprogramować w VBA logikę działania formularzy.
Baza zawiera tabele główne: technicy, wnioskodawcy i punkty świadczenia usług.
